Judicial Auctions Scraper: Pandas DataFrame to Image Bot for Instagram

This program aims to inform individuals about judicial auctions (Argentina), providing insights into available properties and base prices. By increasing awareness, it encourages more interest in these assets, benefiting creditors and auctioning parties in bankruptcy proceedings. Additionally, it aids third parties seeking such information, potentially enabling them to acquire assets of personal interest. The program downloads judicial notices, filters data using web scraping and artificial intelligence, generates a table, which is then visualized as an image with pertinent details. Finally, a bot publishes this image on Instagram.
